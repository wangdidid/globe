The Demo of Observatory
The Demo dynamically maps out the entire world production of goods to create an economic landscape of countries around the globe.
 
The original Intelligarium
The Demo is built upon Intelligarium and Observatory, powerful interactive tools that enables users to visualize a country’s total trade, track how these dynamics change over time and explore growth opportunities for more than a hundred countries worldwide.
For any given country, The Observatory shows which products are produced and exported; The Observatory can then use this information to suggest products a country could begin manufacturing in order to fuel economic growth.
 
Observatory
Observatory is a research tool in Onto' Economic System.
Learn more here: https://observatory.onto.xyz/
 
Data Used
Product and trade data generated with world export data, originally from the United Nations Comtrade database.
 
Technology
This visualization was built with webGL, a new graphics library that enables to create new 3D worlds in the browser. We used Three.js, a javascript library to streamline creating and editing 3D objects.
